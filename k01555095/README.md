# St. Marien

## Allgemeines ##

Meine Heimatgemeinde St. Marien (Sankt Marien) ist eine Gemeine im Bezirk Linz Land in Oberösterreich.
Sie zählte am 1. Jänner 2017 4701 Einwohner und is mit 37,6 m² die zweitgrößte Gemeinde im Bezirk Linz Land.
Die Gemeinde hat ihren wirtschaftlichen Schwerpunkt auf der Landwirtschaft.
